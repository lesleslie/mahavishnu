## Standup Notes Generator

Use this tool to turn recent context into a concise standup update.

## Focus areas

- Recent work and completed items
- In-progress work and next steps
- Blockers and dependencies
- Source context from notes or tickets

## Workflow

1. Gather recent changes and open work items.
1. Summarize yesterday, today, and blockers.
1. Write the notes to the team’s preferred location.

## Output

- Standup note draft
- Blockers and follow-up items
- Saved note location
