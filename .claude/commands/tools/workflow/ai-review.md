## AI/ML Code Review

Use this tool to review AI/ML work for quality, safety, and production readiness.

## Focus areas

- Reproducibility and data leakage
- Prompt safety and token/cost handling
- Model lifecycle and drift monitoring
- Testing and deployment readiness

## Workflow

1. Review model code, data handling, and training split logic.
1. Check prompt security, fallback behavior, and cost controls.
1. Verify model/version management and monitoring hooks.
1. Confirm tests cover preprocessing, pipelines, and edge cases.

## Output

- Severity-ranked review findings
- Improvement recommendations
- Test and rollout checklist
