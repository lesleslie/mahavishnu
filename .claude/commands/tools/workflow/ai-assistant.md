## AI Assistant Development

Use this tool to design an AI assistant with context handling, integrations, and safe execution boundaries.

## Focus areas

- Conversation state and context management
- Intent, entity, and response orchestration
- Tool or API integrations
- Fallbacks, safety, and escalation paths
- Deployment and scaling considerations

## Workflow

1. Define the assistant’s purpose and user journeys.
1. Design the context model and turn-management approach.
1. Choose the integration points and task execution boundaries.
1. Add safety, monitoring, and evaluation loops.
1. Plan deployment and operational upkeep.

## Output

- Assistant architecture outline
- Context and integration plan
- Safety and deployment checklist
