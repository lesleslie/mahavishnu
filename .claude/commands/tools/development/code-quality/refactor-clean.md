## Refactor and Clean Code

Use this tool to improve code quality with practical refactoring steps.

## Focus areas

- Code smells and SOLID violations
- Performance and coupling issues
- Extracting methods and responsibilities
- Test updates after refactors

## Workflow

1. Identify the highest-value code smells and design issues.
1. Choose the smallest refactor that improves clarity or safety.
1. Extract behavior, remove duplication, and simplify flow.
1. Update tests and verify behavior remains intact.

## Output

- Refactoring plan
- Clean-code recommendations
- Test checklist
