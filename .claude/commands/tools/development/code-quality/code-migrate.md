## Code Migration Assistant

Use this tool to plan codebase migrations across frameworks, languages, or major versions.

## Focus areas

- Source analysis and migration complexity
- Risk detection and mitigation
- Phased migration planning
- Rollback and validation
- Parity checks across old and new code paths

## Workflow

1. Analyze the source codebase and target technology.
1. Estimate complexity, risk, and migration effort.
1. Choose a migration strategy and phased rollout.
1. Add validation, parity tests, and rollback steps.
1. Track decommissioning of the old stack.

## Output

- Migration assessment
- Phased migration plan
- Validation and rollback checklist
