## Authentication Implementation

Use this tool to design and implement secure authentication across web, mobile, and API surfaces.

## Focus areas

- Login, registration, and recovery flows
- Session strategy and token handling
- MFA and OAuth/SSO integration
- Route protection and rate limiting
- Frontend and backend integration

## Workflow

1. Pick the auth model that fits the product and threat model.
1. Define session, token, and recovery behavior.
1. Add backend protection, password handling, and validation.
1. Wire frontend state, storage, and redirects.
1. Include monitoring and abuse-prevention controls.

## Output

- Authentication architecture
- Backend and frontend implementation notes
- Security and rollout checklist
