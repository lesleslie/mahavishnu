## API Scaffold Generator

Use this tool to scaffold a production-ready API without overbuilding the first pass.

## Focus areas

- Framework choice: FastAPI, Django REST, Express, or Spring Boot
- Request validation and serialization
- Auth, rate limiting, and config
- Database wiring and migrations
- Tests, docs, and containerization
- Clear module boundaries for services, schemas, and endpoints

## Workflow

1. Pick the framework that fits the stack and team.
1. Define the minimal project structure.
1. Add config, models, endpoints, and service layers.
1. Include tests and deployment basics.
1. Keep generated code aligned with the repository’s conventions.

## Output

- Project skeleton
- Core config and endpoint examples
- Testing and deployment checklist
