## Frontend Bundle Analysis

Use this tool to inspect bundle composition and reduce frontend weight.

## Focus areas

- Bundle size and composition
- Tree-shaking and code splitting
- Lazy loading and asset compression
- Core Web Vitals and runtime performance

## Workflow

1. Measure bundle size and identify large dependencies.
1. Find code-splitting and lazy-loading opportunities.
1. Optimize assets, compression, and critical loading paths.
1. Add performance monitoring and regressions checks.
1. Keep the bundle strategy aligned with the app’s architecture.

## Output

- Bundle analysis summary
- Optimization recommendations
- Monitoring checklist
