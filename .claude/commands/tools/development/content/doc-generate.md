## Automated Documentation Generation

Use this tool to generate and maintain code-aware documentation.

## Focus areas

- API and schema extraction
- Architecture and user guide generation
- Examples and references that stay aligned with code
- Diagram and OpenAPI support

## Workflow

1. Inspect the source code and identify public surfaces.
1. Extract endpoints, models, and key concepts.
1. Generate concise docs and diagrams from the analysis.
1. Keep the docs synchronized with code changes.
1. Prefer short living docs over large static manuals.

## Output

- Documentation outline
- Example docs or diagram plan
- Sync and maintenance checklist
