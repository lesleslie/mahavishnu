## Workflow Orchestrator

Use this tool to coordinate workflows that span multiple apps, windows, or execution contexts.

## Focus areas

- Cross-application coordination
- Window and tab management
- User prompts and decision points
- State preservation and verification

## Workflow

1. Identify the workflow goal and execution context.
1. Decide which windows, tabs, or apps need coordination.
1. Run steps with verification and graceful fallback.
1. Keep state and context preserved across phases.

## Output

- Workflow plan
- Coordination and verification notes
- Rollback or fallback checklist
