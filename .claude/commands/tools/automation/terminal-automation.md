## Terminal Automation

Use this tool to automate terminal sessions, tabs, and workflow setup in a controlled way.

## Focus areas

- Window and tab management
- Session coordination across projects
- Reproducible terminal layouts
- Verification of automated actions

## Workflow

1. Identify the terminal task and target session.
1. Create or update the minimal window/tab layout needed.
1. Run commands with verification and recovery in mind.
1. Keep the session organization aligned to the project.

## Output

- Terminal session plan
- Window/tab layout notes
- Verification checklist
