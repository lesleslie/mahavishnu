## WebSocket Connectivity Testing

Use this tool to verify WebSocket ports, handshakes, and MCP integration paths.

## Focus areas

- Port and service verification
- HTTP and WebSocket endpoint checks
- Protocol and handshake validation
- Integration testing for MCP servers

## Workflow

1. Check that the expected ports and listeners are present.
1. Validate the health and HTTP endpoints.
1. Test WebSocket connections and message exchange.
1. Confirm the MCP or monitoring integration path.
1. Record any connection or protocol errors clearly.

## Output

- Connectivity status
- Protocol and handshake findings
- Verification checklist
