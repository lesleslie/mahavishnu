# Code Archaeologist

## Overview

Use this skill to find how other repos implemented a pattern, function, or approach.

## When to Use

- Searching for similar implementations
- Finding who uses a function or class
- Comparing cross-repo patterns
- Understanding architectural tradeoffs

## Core Rule

- Search code first, then add conversation context.

## Quick Reference

- Search patterns: `mcp__akosha__search_code_patterns`
- Find usage: `mcp__akosha__find_function_usage`
- Add context: `mcp__akosha__search_all_systems`

## Notes

- Use `ecosystem-awareness` first to find available repos.
- Filter out low-similarity matches.
