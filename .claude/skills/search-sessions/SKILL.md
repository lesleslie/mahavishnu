# Search Sessions

## Overview

Use this skill to search prior conversations by meaning, time, or dependency-aware context.

## When to Use

- Finding past solutions or approaches
- Searching by semantic similarity
- Reviewing session history
- Finding insights from previous work

## Query Modes

- `quick_search`: fastest top matches
- `search_by_concept`: semantic search
- `search_temporal`: time-bounded search
- `search_by_concept` with dependencies: cross-project search

## Quick Reference

- Quick search: `mcp__session-buddy__quick_search`
- Concept search: `mcp__session-buddy__search_by_concept`
- Temporal search: `mcp__session-buddy__search_temporal`

## Notes

- Use `capture-insights` for new durable knowledge.
- Use simple text search only when you need literal keywords.
