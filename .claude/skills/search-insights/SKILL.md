# Search Insights

## Overview

Use this skill for semantic search, trend analysis, anomaly detection, and knowledge graph queries.

## When to Use

- Searching for past solutions or patterns
- Analyzing trends across sessions
- Finding entity relationships and connected concepts
- Detecting anomalies in workflow metrics

## Query Modes

- `semantic`: search by meaning, not keywords
- `time-series`: trends, correlations, and anomalies
- `graph`: entity relationships and paths

## Quick Reference

- Search: `mcp__akosha__search_all_systems`
- Trends: `mcp__akosha__analyze_trends`
- Graph: `mcp__akosha__query_knowledge_graph`
- Anomalies: `mcp__akosha__detect_anomalies`

## Notes

- Use `capture-insights` when you are creating new durable knowledge.
- Use session search for quick lookups and Akosha for cross-system queries.
